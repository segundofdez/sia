# SIA

Sia is a (css | less) library to make simple icon animation.

## How to use
* Import the css file into your html:
  ```html
    <link rel="stylesheet" href="css/sia.min.css">
  ```
* Add the html code:
```html
<div class="sia">
    <a class="sia-nav" href="#"><span>toggle navigation</span></a>
</div>
```

* Choose your default icon:

  * `sia-nav`
  * `sia-cross`
  * `sia-plus`
  * `sia-minus`
  * `sia-check`
  * `sia-search`
  * `sia-top`
  * `sia-right`
  * `sia-bottom`
  * `sia-left`
  * `sia-toparrow`
  * `sia-rightarrow`
  * `sia-bottomarrow`
  * `sia-leftarrow`
  * `sia-circle`
  * `sia-circleo`
